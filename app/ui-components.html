Buttons

Naked Button

//als er een block component is kan elke button met uitleg los in een 'block'
The naked button is used blabla
The 'content' property describes the action..
The 'class' property defines the button style. Possible options are:

  • naked-button--neutral
  • naked-button--happy
  • naked-button--strict
  • naked-button--angry
  • pull-left/right
  • 'disabled':'true'

  $(ui.component.button.render({
    'content' : 'Neutral',
    'class'   : 'naked-button--neutral'
  }))

Messages

Alerts

You can either use the solid or the naked alert.
The 'class' property defines the alert style. Possible options are:

  • naked-alert--happy
  • naked-alert--strict
  • naked-alert--angry

$(ui.component.alert.render({
  'icon': 'icon-user-award',
    'title': 'YES!',
    'class': 'solid-alert--happy',
    'message': 'You did something right',
    'button': [{
        'iconOnly': true,
        'icon': 'icon-system-close',
        'class': 'naked-button--icon-only--happy'
    }]
}))
