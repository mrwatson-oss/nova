Buttons

Naked Button

//als er een block component is kan elke button met uitleg los in een 'block'
The naked button is used blabla
The 'content' property describes the action..
The 'class' property defines the button style. Possible options are:

  • naked-button--neutral
  • naked-button--happy
  • naked-button--strict
  • naked-button--angry
  • pull-left/right
  • 'disabled':'true'

  $(ui.component.button.render({
    'content' : 'Neutral',
    'class'   : 'naked-button--neutral'
  }))
