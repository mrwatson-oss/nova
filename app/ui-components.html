  • naked-alert--strict
  • naked-alert--angry

Messages

Alerts

You can either use the solid or the naked alert.
The 'class' property defines the alert style. Possible options are:

  • naked-alert--happy
  • naked-alert--strict
  • naked-alert--angry

$(ui.component.alert.render({
  'icon': 'icon-user-award',
    'title': 'YES!',
    'class': 'solid-alert--happy',
    'message': 'You did something right',
    'button': [{
        'iconOnly': true,
        'icon': 'icon-system-close',
        'class': 'naked-button--icon-only--happy'
    }]
}))
